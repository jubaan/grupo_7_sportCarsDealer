### How to get started

1. Clone repository
   ```bash
   git clone https://github.com/jubaan/grupo_7_sportCarsDealer.git
   ```
2. Change directory
   ```bash
   cd grupo_7_sportCarsDealer
   ```
3. Install dependencies
   ```bash
   npm i
   ```
4. Start project
   ```bash
   npm start
   ```
5. Visit `localhost`
   ```bash
   http://localhost:3000/
   ```
